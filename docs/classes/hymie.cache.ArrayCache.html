\hymie\cache

\hymie\cacheArrayCache

简单的符合 psr-16 标准的数组缓存实现。

数组结构为： [key] => ['value' => value, 'expire'=>['ttl'=> ttl, expire_at => expire-at];

缓存 ttl 以秒为单位。
