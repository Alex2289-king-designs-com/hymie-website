getController()

getController() : array

根据路由配置获取控制器，控制器方法的 Reflection 实例。

如果找不到控制器则抛出 RouterException，最终会触发 404 页面。

返回的控制器信息数组格式为:

array( 'module' => $module, // 模块名 'controller' => $controller, // 控制器对象 'method' => $reflectMethod, // 方法放射对象 'params' => $paramsArray); // 参数数组

array
